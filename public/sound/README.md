# Sound Files

⚠️ **LƯU Ý**: Thư mục này đã được đổi tên từ `sounds` sang `sound`

Đường dẫn sử dụng trong code: `/sound/success.mp3`

Thêm file âm thanh vào thư mục `public/sound/`:

- `success.mp3` - Âm thanh chúc mừng khi hoàn thành bài tập

Bạn có thể tải âm thanh miễn phí từ:

- https://mixkit.co/free-sound-effects/
- https://freesound.org/
- https://pixabay.com/sound-effects/

Hoặc tạo file âm thanh đơn giản bằng các công cụ online.
